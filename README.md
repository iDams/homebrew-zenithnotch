# ZenithNotch

![ZenithNotch Logo](logo.png)

Early Homebrew Cask distribution for **ZenithNotch** on macOS.

## Install

```bash
brew tap iDams/zenithnotch
brew install --cask zenithnotch
```

## Current Status

This is an **Alpha / Experimental** release.

- Features may be incomplete.
- Some behaviors may be unstable.
- Breaking changes can happen between versions.
- Bugs are expected while the project is still very early.

If you want production-level stability, wait for a later stable release.

## Update

```bash
brew update
brew upgrade --cask zenithnotch
```

## Uninstall

```bash
brew uninstall --cask zenithnotch
```

## Notes

- Cask file path: `Casks/zenithnotch.rb`
- This repository is focused on distribution via Homebrew Cask.
